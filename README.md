# Landing Page

A landing page built as part of [The Odin Project]
Foundations course — practicing Flexbox for layout.

🔗 [Live Demo]: https://farhanjawed1222.github.io/landing-page-project/

## What I practiced
- Flexbox (main axis, cross axis, flex-grow/shrink/basis)
- Building a nav bar, hero section, and footer with Flexbox
- Structuring a multi-section page with semantic HTML

## Sections
- Header/Nav
- Hero
- Content (image + text rows)
- Call to Action
- Footer

## Note
This project is not yet responsive — media queries and mobile layout 
will be covered in a later section of the curriculum.
